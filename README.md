# Porcupine

A key-value store cli app

```zsh
porcupine set hello world
porcupine get hello
porcupine set foo bar
porcupine ls
world
bar
```

## Commands

```zsh
porcupine get  get a value by key
porcupine set  set key to a value
porcupine ls   list stored keys
porcupine rm   remove an entry by key
```
